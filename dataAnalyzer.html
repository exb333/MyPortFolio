DATA ANALYZER APP

LOGIN Screen

When user opens the app, the first window that pops up is login screen. In this the user enters correct datameer and oracle credentials

Welcome Window

As you can see this is the welcome screen of the app, it contains three different apps. We will start here with Result Graph.

WorkBook Selection Window

Here we enter the workbook ID obtained from datameer. After providing workbook ID we get the list of sheets available in that workbook. Now I have selected result_cnt sheet in order to visualize data.

Visualization Window

Here We select the test numbers from the list available at the top left. As soon as we select test numbers, the scatter graph below changes dynamically according to the data available for the test numbers.

Categorical Data Visualization

If you look at the previous screen, there is a table on top right which displays Categorical Data and next to it at left there is a button which opens this window if clicked.

Welcome Window

Again welcome screen of the app, this time we will see Trends To Oracle app.

Data Migration Window

Some users are restricted to access data from SQL Server, so for this reason this app was designed, the purpose of this app is to migrate data from SQL Server to Oracle and make specific data available to the restricted users.
